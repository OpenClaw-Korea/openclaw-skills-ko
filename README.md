## 🚀 설치

```bash
# ClawHub에서 설치 (출시 후)
npx clawhub@latest install openclaw-skills-ko/<skill-name>

# 또는 수동 복사
cp -r skills/<skill-name> ~/openclaw/skills/
```

## 🤝 기여하기

한국 서비스 연동 스킬을 만들어보세요!

1. Fork → Branch → PR
2. `skills/<skill-name>/SKILL.md` 필수
3. 한국어 설명 포함

### 스킬 아이디어 (PR 환영!)

- 📰 한국 뉴스 브리핑

- 🎵 멜론/지니 차트
