# 배달의민족

배달의민족(배민) 상점 및 메뉴 정보를 조회하는 스킬입니다.

> ⚠️ **주의:** 배민은 공식 공개 API를 제공하지 않습니다. 이 스킬은 배민 웹/앱에서 사용하는 내부 API 엔드포인트를 활용하며, 언제든 변경될 수 있습니다.

## 현실적 기능 범위

### ✅ 가능한 것
- **배민 상점 검색** — 배민 웹사이트에서 키워드로 상점 검색
- **상점 메뉴 조회** — 특정 상점의 메뉴/가격 정보 파싱
- **인기 카테고리 조회** — 치킨, 피자, 중식 등 카테고리별 탐색

### ⚠️ 제한적인 것
- **주문하기** — 인증/결제 필요, 자동화 불가
- **실시간 배달 추적** — 로그인 세션 필요
- **리뷰 작성** — 인증 필요

### 🔧 접근 방식
1. **배민 웹 (baemin.com)** 페이지 파싱
2. **배민 내부 API** 리버스 엔지니어링 (비공식, 변경 가능)
3. **배민 상점 페이지** HTML 스크래핑

## 설치

```bash
# 필요: curl, jq, (선택) pup (HTML 파서)
brew install jq
# pup 설치 (HTML 파싱용)
go install github.com/ericchiang/pup@latest
```

## 환경변수

| 변수 | 설명 | 필수 |
|------|------|------|
| `BAEMIN_LATITUDE` | 배달 위치 위도 (예: 37.5665) | ✅ |
| `BAEMIN_LONGITUDE` | 배달 위치 경도 (예: 126.9780) | ✅ |

## 사용법

### 상점 검색
```bash
./search.sh "치킨" [결과수]
```

### 인기 카테고리 목록
```bash
./search.sh --categories
```

## 예시

```bash
# 근처 치킨집 검색
export BAEMIN_LATITUDE=37.5665
export BAEMIN_LONGITUDE=126.9780
./search.sh "치킨"

# 카테고리 목록
./search.sh --categories

# 피자집 상위 5개
./search.sh "피자" 5
```

## 참고

- 이 스킬은 비공식 방법을 사용하므로 배민 측 변경에 따라 동작하지 않을 수 있습니다
- 과도한 요청은 IP 차단의 원인이 될 수 있으니 주의해주세요
- 배민 이용약관을 확인하고 개인적 용도로만 사용해주세요
